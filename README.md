🏡 California Housing Price Prediction using Linear Regression
📊 Machine Learning Project using Python & Scikit-Learn

✨ Beginner Friendly • 📈 Data Visualization • 🤖 Linear Regression • 📚 Educational Project

📖 Overview

This project demonstrates a simple Machine Learning workflow using Linear Regression on the California Housing dataset.

The notebook covers:

Loading and understanding the dataset
Data preprocessing and analysis
Correlation analysis
Data visualization
Splitting data into training and testing sets
Training a Linear Regression model
Evaluating model performance

This project is beginner-friendly and helps in understanding the basics of supervised machine learning.

🛠️ Technologies Used
Python
Pandas
NumPy
Matplotlib
Scikit-learn
📂 Dataset

The project uses the California Housing Dataset provided by Scikit-learn.

Features included in the dataset:

MedInc
HouseAge
AveRooms
AveBedrms
Population
AveOccup
Latitude
Longitude

Target Variable:

House Price
⚙️ Project Workflow
🔹 1. Import Libraries

Required Python libraries are imported for data handling, visualization, and machine learning.

🔹 2. Load Dataset

The California Housing dataset is loaded using:

from sklearn.datasets import fetch_california_housing
🔹 3. Exploratory Data Analysis

The notebook checks:

Dataset shape
Missing values
Statistical summary
Correlation between features
🔹 4. Data Visualization

Visualizations included:

Correlation heatmap
House price distribution histogram
🔹 5. Train-Test Split

The dataset is divided into:

Training Data (80%)
Testing Data (20%)
🔹 6. Model Training

A Linear Regression model is trained using:

from sklearn.linear_model import LinearRegression
🔹 7. Model Evaluation

The model is evaluated using:

Mean Absolute Error (MAE)
Mean Squared Error (MSE)
Root Mean Squared Error (RMSE)
R² Score
